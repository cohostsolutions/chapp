# AI Training Simulator

This feature enables customer service reps and sales agents to practice conversations with AI personas and receive feedback.

## Overview

- Scenario-based roleplay with configurable persona, industry, and difficulty.
- Real-time coaching hints for better questioning, empathy, and clarity.
- Post-session evaluation generated by AI, with strengths and improvements.
- Local session persistence (upgradeable to Supabase-backed storage).

## Files

- `src/pages/AITraining.tsx`: Main page to select scenarios and run sessions.
- `src/components/training/TrainingSimulator.tsx`: Chat simulator that primes the AI with persona instructions.
- `src/components/training/LiveFeedbackPanel.tsx`: Lightweight coaching hints during the session.
- `src/components/training/SessionSummary.tsx`: Displays AI evaluation and category scores.
- `src/lib/training/scenarios.ts`: Example scenarios for customer service and sales.
- `src/lib/training/types.ts`: Shared types.
- `src/lib/training/storage.ts`: LocalStorage persistence for session records.

## How it works

The simulator uses the existing `useUnifiedAI` hook to invoke the Supabase `ai-chat` function. It primes the conversation by sending a "persona primer" message that instructs the AI to roleplay a specific customer persona. At the end of a session, it asks the AI to evaluate performance and attempts to parse JSON. If parsing fails, the raw text is shown.
