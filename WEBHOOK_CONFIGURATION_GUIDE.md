### Updating Verify Token in Your Code

If needed, add webhook verification to your Supabase function:

```typescript
// In supabase/functions/social-webhook/index.ts
// Add this near the top after other environment variables:

const WEBHOOK_VERIFY_TOKEN = Deno.env.get('WEBHOOK_VERIFY_TOKEN') || '';

// Then add verification challenge handler:
if (req.method === 'GET') {
  const url = new URL(req.url);
  const mode = url.searchParams.get('hub.mode');
  const token = url.searchParams.get('hub.verify_token');
  const challenge = url.searchParams.get('hub.challenge');
  
  if (mode === 'subscribe' && token === WEBHOOK_VERIFY_TOKEN) {
    console.log('Webhook verified');
    return new Response(challenge, { status: 200 });
  } else {
    return new Response('Forbidden', { status: 403 });
  }
}
```

## Common Issues

### Issue 1: "Webhook Verification Failed"

**Cause**: Verify token mismatch
**Solution**: 
1. Check what token you entered in Facebook
2. Make sure it matches your environment variable
3. Redeploy function if needed

### Issue 2: "Invalid Signature"

**Cause**: META_APP_SECRET incorrect or missing
**Solution**:
```bash
# Verify secret is set
supabase secrets list

# Set it if missing (use App Secret from Facebook App Dashboard)
supabase secrets set META_APP_SECRET=your_app_secret_from_facebook

# Redeploy
supabase functions deploy social-webhook
```

### Issue 3: "Webhook Unreachable"

**Cause**: Function not deployed or URL incorrect
**Solution**:
```bash
# Check function exists
supabase functions list

# Deploy if needed
supabase functions deploy social-webhook

# Test URL
curl https://domipubyjkhsrmdwtabh.supabase.co/functions/v1/social-webhook
```

## OAuth vs Webhook

**Different URLs for different purposes:**

| Purpose | URL | Used For |
|---------|-----|----------|
| **OAuth Callback** | `/facebook-callback` | User authorization, getting tokens |
| **Webhook** | `/functions/v1/social-webhook` | Receiving messages from Meta |

- OAuth callbacks go through your **React app** routes
- Webhooks go directly to your **Supabase Edge Function**
